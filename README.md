# 数学教材辅导 Skill

> 一个面向数学基础薄弱学习者的大学数学教学方法论 —— 把抽象数学用大白话 + 生活比喻讲清楚。

## 这是什么？

这是一个为 AI 助教（Claude Code Skill）设计的教学指南。它定义了一套系统化的数学教学方法，让 AI 能够像耐心的家教一样，帮助数学基础薄弱的学习者（大学文科生水平）逐章学习大学数学教材。

## 核心理念

**先讲"为什么重要 / 直觉是什么"，再讲"数学怎么写"，绝不反过来。**

每块内容按五步讲解：

```
① 大白话直觉  →  ② 生活比喻  →  ③ 符号解释  →  ④ 数学本身  →  ⑤ 一句话总结
```

## 适用场景

- 学习者上传数学教材 PDF，说"帮我讲这一章"
- 想系统学习凸优化、线性代数、微积分等大学数学
- 数学基础薄弱（文科生水平），需要大白话解释

## 已验证的教学内容

基于 Stephen Boyd & Lieven Vandenberghe《Convex Optimization》全书教学实践验证，涵盖：

- 凸集、凸函数基础（第 1-3 章）
- 凸优化问题：LP / QP / SOCP / SDP 问题家族（第 4 章）
- 对偶理论：拉格朗日对偶、KKT 条件、敏感性分析（第 5 章）

## 使用方法

1. 将本目录作为 Claude Code Skill 加载
2. 上传数学教材 PDF
3. 告诉 AI 你要学哪一章 / 哪一页
4. AI 会按 Skill 规范分块教学，每块讲完等你确认再继续
