# 凸优化全书知识图谱

本文件记录《凸优化》（Boyd & Vandenberghe）各章核心内容、概念关系、以及已验证的教学重难点。

---

## 全书结构

```
第1-3章：基础工具（凸集、凸函数、共轭函数等）
第4章：凸优化问题（标准形式、标准问题家族、变换技巧）
第5章：对偶理论（拉格朗日、KKT、敏感性分析）
第6-9章：算法（下降法、内点法等）
```

---

## 第四章：凸优化问题

### 核心主线
```
优化问题三要素
    ↓
凸优化的特殊性：局部最优=全局最优
    ↓
标准问题家族：LP ⊂ SOCP ⊂ SDP
    ↓
变换技巧：把非凸变成凸
    ↓
DCP/CVXPY：从理论到实践
    ↓
多目标优化：帕累托+标量化
```

### 关键结论
1. 凸优化问题的局部最优就是全局最优（反证法证明）
2. 标准问题家族：LP ⊂ QP ⊂ QCQP ⊂ SOCP ⊂ SDP，LP和SOCP都是SDP特例
3. 凸松弛：把{0,1}放宽成[0,1]，离散问题变LP
4. 多目标优化：帕累托曲线+标量化，Ridge回归和有效前沿都是标量化的应用

### 已验证的教学难点
- **变换技巧**：学习者最容易失去方向感，每讲一个技巧要说清楚"这个技巧适用于什么情形"
- **切比雪夫中心推导**：aᵢᵀxc + r‖aᵢ‖₂ ≤ bᵢ 需要配合柯西-施瓦茨仔细推导
- **多目标和标量化的联系**：要明确说"γ就是帕累托权重"才能让学习者明白Ridge回归的意义

### 比喻对应
- LP可行域：多面体/多边形
- QP：在多面体上扣一个碗
- SOCP处理不确定性：最坏情况+安全缓冲
- SDP：矩阵不等式，"一口气约束所有方向"
- 帕累托曲线：手机续航vs重量，投资收益vs风险

## 第五章：对偶理论

### 核心主线
```
拉格朗日函数（把约束融进目标）
    ↓
对偶函数（L关于x的最小值，永远≤p*）
    ↓
对偶问题（最大化g，找最好的下界）
    ↓
弱对偶（d*≤p*，永远成立）
    ↓
强对偶（d*=p*，凸问题+Slater条件）
    ↓
互补松弛（强对偶推出λᵢfᵢ=0）
    ↓
KKT条件（四个条件，充要判断最优）
    ↓
敏感性分析（λ*是影子价格）
```

### 关键结论
1. g(λ,ν) ≤ p*，永远成立（弱对偶），是铁律
2. 对偶问题永远是凸优化，不管原问题是否凸
3. LP的对偶是LP，QP的对偶是QP
4. Slater条件（严格可行）→ 强对偶成立
5. KKT四个条件：主可行、对偶可行、互补松弛、梯度为零
6. λᵢ* = -∂p*/∂uᵢ（影子价格）
7. Farkas引理：两系统恰好一个有解
8. 无套利定理：无套利 ⟺ 存在风险中性概率

### 已验证的教学难点（按难度排序）

**极难：**
- 对偶函数的下界性质证明：要从"可行点满足约束→L≤f₀→g≤f₀"一步步讲
- 互补松弛推导：等式链"f₀=g=inf L ≤ L(x*) ≤ f₀"学习者经常问"为什么一串不等式最后等于原式有什么用"，需要解释"夹逼"的逻辑
- KKT的意义：学习者容易误以为KKT只是重复了强对偶，要强调KKT是"检验清单"，解决的是"怎么验证某个点是不是最优"这个不同的问题

**较难：**
- 扰动问题推导：-uᵀλ* - vᵀν* 怎么来的（扰动改变了L，带到了对偶函数里）
- Farkas引理证明：用LP强对偶，系统二恰好是对偶的可行性条件

**中等：**
- 对偶问题的定义（很多学习者第一次听觉得自然）
- 敏感性分析的经济意义（影子价格直觉很好理解）

### 符号易混淆点
- λ（拉格朗日乘子）和 λ（特征值）：第五章全部是前者
- ν（对偶变量）读作"nu"，不要读成"v"
- p*（主问题最优值）vs d*（对偶问题最优值）：两者关系是核心

## 学习者常见误解

| 误解 | 正确理解 |
|---|---|
| "KKT就是在说强对偶" | KKT是验证工具（给定一个点，判断它是不是最优），强对偶是关于最优值相等的结论 |
| "对偶问题的解就是主问题的解" | 强对偶只说最优值相等，解可以完全不同 |
| "λ≥0是人为规定的" | λ≥0是从"保证下界"的逻辑必然推出来的 |
| "互补松弛是第三个约束" | 互补松弛是从强对偶推导出来的结论，不是额外约束 |
| "Farkas引理很难" | 用具体数字例子（一维情形）往往立刻就懂了 |
