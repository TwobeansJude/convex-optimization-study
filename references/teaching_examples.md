# 已验证有效的教学案例库

本文件收录在实际教学过程中被证明有效的解释方式、类比和数字例子。  
遇到对应概念时优先使用这些案例，不要重新发明。

---

## 1. 优化问题三要素

**有效比喻：外卖配送规划**
- x = 路线选择
- f₀(x) = 总配送距离（要最小化）
- fᵢ(x) ≤ 0 → 车辆不超载
- hᵢ(x) = 0 → 每个订单必须送到

**教学验证：** 这个例子在第一页就建立了直觉，后续所有概念都可以回到这个场景。

---

## 2. 局部最优 vs 全局最优

**有效比喻：山地徒步找最低点**
- 你走到一个小山谷，四面都比你高，以为到底了
- 但远处还有更深的山谷
- 非凸问题的"假山谷" = 局部最优
- 凸优化的保证：不存在假山谷

**有效数字例子：** f(x) = x³ - 3x
- x=1 是局部最优（附近都比它大）
- 但 x→-∞ 时函数值→-∞，所以 p* = -∞
- 这说明局部最优 ≠ 全局最优

## 3. 对数障碍函数

**核心直觉：** b - aᵀx 越接近0，惩罚越大（趋向-∞）
- 不是硬墙（硬性禁止越界）
- 是软墙（靠近就越烫手）

**有效比喻：开车靠近悬崖**
- 方法一（硬约束）：立墙，碰到就停
- 方法二（对数障碍）：地面越靠近边缘越陡，油耗指数暴增，自然远离

**为什么用对数而不是其他函数：** 趋近0时→-∞，远处增长平缓不过分干扰。

## 4. 拉格朗日函数

**核心直觉：** 把约束"融进"目标函数
- 原来：在约束范围内找最优
- 拉格朗日：x 自由移动，但违反约束要付代价
- λᵢ 是"罚款金额"——违反越多，罚越多

**为什么 λ ≥ 0：** 可行点满足 fᵢ(x) ≤ 0。要让惩罚项 λᵢfᵢ(x) ≤ 0（即 L ≤ f₀），必须 λᵢ ≥ 0。这是逻辑推出来的，不是规定。

---

## 5. 对偶函数下界

**有效解释方式（不用图，纯文字推导）：**

1. 可行点 x̃ 满足 fᵢ(x̃) ≤ 0，hᵢ(x̃) = 0
2. 代入 L：L(x̃,λ,ν) = f₀(x̃) + ∑λᵢfᵢ(x̃) + ∑νᵢhᵢ(x̃)
3. 等式约束项 = 0（因为 hᵢ(x̃)=0）
4. 不等式约束项 ≤ 0（因为 λᵢ≥0 且 fᵢ(x̃)≤0）
5. 所以 L(x̃,λ,ν) ≤ f₀(x̃)
6. g(λ,ν) = inf L ≤ L(x̃,λ,ν) ≤ f₀(x̃)
7. 对所有可行点 x̃ 成立 → g(λ,ν) ≤ p*

**注意：** 买东西的比喻（"价格下限估算"）对这个概念很有用，但要在推导完之后再用，不要替代推导。

## 6. 互补松弛推导

**最容易让学习者懵的问题：** "为什么推出 f₀(x*) ≤ f₀(x*) 有意义？"

**正确解释方式：**
整条链是：A = B = C ≤ D ≤ A，其中 A = f₀(x*)。
一个数既然 ≤ 自身，说明链里每个 ≤ 都取等。
这就是"夹逼"——不是推出了废话，是推出了每个不等号都是等号。

然后从"L(x*,λ*,ν*) = f₀(x*)"推出 ∑λᵢ*fᵢ(x*) = 0。
再从"每项 ≤ 0 但加起来 = 0"推出每项 = 0。

## 7. KKT的意义

**学习者最常见的误解：** "KKT只是在重复强对偶，有什么新东西？"

**正确解释：**
- 强对偶说"p* = d*"——这是关于**最优值**的结论
- KKT说"满足这四条的点就是最优点"——这是关于**验证某个点**的工具
- 两者解决不同问题：一个说值是多少，一个说怎么验证

**有效类比：**
- 强对偶：这道题满分是100分
- KKT：你怎么知道自己拿了满分？（检查清单）

## 8. 影子价格

**有效数字例子：工厂用电规划**
- 约束：每天用电 ≤ 1000度
- 最优利润：50万
- λ* = 0.3（万元/度）

含义：
- 电力配额放宽1度 → 利润提升约0.3万元
- 如果电价 < 0.3万元/度，买电合算
- 如果电价 > 0.3万元/度，不值得

## 9. Farkas引理

**最有效的解释方式：** 用一维具体数字

设 A = [[1]]（只有一行 a = [1]），c = [-1]，x 是实数。

系统一：Ax ≤ 0 且 cᵀx < 0
- x ≤ 0 且 (-1)x < 0 即 x > 0
- 矛盾，**无解**

系统二：Aᵀy + c = 0，y ≥ 0
- y + (-1) = 0 → y = 1 ≥ 0 ✓
- **有解**

换 c = [2]：
系统一：x ≤ 0 且 2x < 0，取 x = -1 满足，**有解**
系统二：y + 2 = 0 → y = -2 < 0，**无解**

这两个例子展示了"恰好一个有解"。

## 10. 无套利定理

**从生活出发的解释：**
同一瓶水，便利店1元，超市3元 → 套利（无风险赚钱）

**正式定义的关键点：**
- pᵀx < 0：现在收到钱
- Vx ≥ 0：未来任何结局都不亏

套利 = Farkas系统一，风险中性概率 = Farkas系统二，定理由Farkas引理直接得出。

**192页数字例子的教学重点：**
第一组价格（p = [1.0, 0.9, 0.3]）：套利存在，因为股票A被低估（价格0.9但期望回报>0.9）
第二组价格（p̃ = [1.0, 0.8, 0.7]）：无套利，验证：y=[0.36,0.27,0.26,0.11]加总=1且Vᵀy=p̃

## 11. 凸优化vs非凸问题的本质区别

**有效总结方式：**
- 非凸问题：可能有假山谷，算法可能被困住，找到局部最优不代表找到全局最优
- 凸优化：不存在假山谷，找到就是真正的最优，可靠高效

**这就是为什么凸优化重要：** 不是因为它限制了问题的范围，而是因为它给了"找到的就是最好的"这个保证。

## 12. 帕累托最优和标量化

**帕累托的核心直觉：**
手机A续航12小时重180g，手机C续航14小时重200g。
A更轻，C续航更长，各有优势，互不支配，两者都是帕累托最优。

**标量化的核心直觉：**
λ₁×续航得分 + λ₂×重量得分
λ₂大 = 你更在乎轻 → 选A
λ₁大 = 你更在乎续航 → 选C

**Ridge回归的联系（很多学习者惊喜的发现）：**
minimize ‖Ax-b‖² + γ‖x‖²
这里的 γ 就是帕累托权重，连接了凸优化和机器学习。
