# 数学符号速查表

本表列出教学中最常出现的符号，以及向数学基础薄弱的学习者解释时的标准用语。

---

## 基础符号

| 符号 | 标准解释 | 注意事项 |
|---|---|---|
| ∑ | 连加符号，把后面的东西算很多次全部加起来 | 要配合说清楚"i从几到几" |
| inf | 下确界，可以理解为"最小值（或趋近的最小值）" | 区别于min：inf可以取不到 |
| sup | 上确界，可以理解为"最大值（或趋近的最大值）" | 同上 |
| ∈ | "属于"，x∈A 读作"x是A里的一个元素" | |
| ∩ | 交集，两个集合共同有的部分 | 用"两个圆的重叠部分"类比 |
| ∪ | 并集，两个集合合在一起 | |
| ∀ | "对所有"，∀x读作"对任意的x" | |
| ∃ | "存在"，∃x读作"存在一个x" | |
| → | "趋向"，x→∞读作"x趋向无穷大" | |
| ≤ / ≥ | 小于等于 / 大于等于 | |
| ≼ | 矩阵意义下的"小于等于"，半负定 | 区别于标量的≤ |
| ≽ | 矩阵意义下的"大于等于"，半正定 | |

## 向量和矩阵符号

| 符号 | 标准解释 | 示例说明 |
|---|---|---|
| xᵀ | x的转置，把列向量变成行向量 | [1,2,3]转置后可以和矩阵相乘 |
| xᵀy | 两个向量的内积，对应位置相乘再加总 | [1,2]ᵀ[3,4] = 1×3+2×4 = 11 |
| Ax | 矩阵A乘以向量x，线性变换 | 把x映射到新的向量 |
| ‖x‖₂ | x的欧几里得长度，√(x₁²+x₂²+…) | 就是勾股定理推广到多维 |
| ‖x‖₁ | x的L1范数，所有分量绝对值之和 | |x₁|+|x₂|+… |
| ‖x‖∞ | x的无穷范数，最大分量的绝对值 | max(|x₁|,|x₂|,…) |
| ‖x‖* | 对偶范数，配套原范数的另一种范数 | L2的对偶是L2，L1的对偶是L∞ |
| diag(v) | 以向量v为对角线的矩阵，其余位置是0 | v=[1,2,3]→3×3对角矩阵 |
| Aᵀ | 矩阵A的转置，行列互换 | |
| A⁻¹ | 矩阵A的逆矩阵 | AA⁻¹=I（单位矩阵） |
| A† | 矩阵A的伪逆 | 推广的逆矩阵，适用于非方阵 |
| I | 单位矩阵，对角线全是1其余是0 | |

## 微积分符号

| 符号 | 标准解释 | 示例说明 |
|---|---|---|
| ∇f(x) | f关于x的梯度，多维版的导数 | 指向函数值增长最快的方向 |
| ∇²f(x) | f的Hessian矩阵，二阶导数推广 | 描述函数的弯曲程度 |
| ∂f/∂xᵢ | f对第i个分量的偏导数 | 固定其他分量，只对xᵢ求导 |
| df | 函数的微分，无穷小的变化量 | |

## 凸优化专用符号

| 符号 | 标准解释 | 注意事项 |
|---|---|---|
| dom f | f的定义域，f能接受的x的范围 | log(x)的dom是x>0 |
| p* | 主问题的最优值 | |
| d* | 对偶问题的最优值 | |
| λ | 不等式约束的拉格朗日乘子，必须≥0 | |
| ν（nu）| 等式约束的拉格朗日乘子，无符号限制 | |
| L(x,λ,ν) | 拉格朗日函数 | |
| g(λ,ν) | 对偶函数，L关于x的最小值 | |
| Sₙ₊ | n×n半正定矩阵的集合 | "碗不倒扣"的矩阵 |
| Sₙ₊₊ | n×n正定矩阵的集合 | 严格的碗形 |
| Rₙ₊₊ | n维严格正实数向量 | 所有分量都>0 |
| epi f | f的上图，函数图像上方的所有点 | |
| f*(v) | f的共轭函数，sup_y[vᵀy-f(y)] | 对偶推导中自然出现 |

## 概率相关符号（第五章替代定理部分）

| 符号 | 标准解释 |
|---|---|
| N(μ,Σ) | 均值为μ、协方差矩阵为Σ的正态分布 |
| Φ(u) | 标准正态分布的累积分布函数 |
| E[X] | 随机变量X的期望值 |

## 符号使用注意事项

1. **同一符号在不同上下文含义不同**
   - ≤：数的小于等于 vs 矩阵的半负定（≼）
   - λ：拉格朗日乘子 vs 特征值（教材里一般不混用但要注意）

2. **第一次出现必须解释**  
   不要假设学习者知道任何符号，哪怕是∑这样的基础符号

3. **用数字例子配合解释**  
   光说定义不够，一定要给一个2-3维的具体数字例子
