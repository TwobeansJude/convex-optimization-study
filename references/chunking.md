# 分块策略详细指南

## 核心原则

**宁可分太细，不能分太粗。**  
学习者跟不上比讲得慢更糟糕。一块内容讲完学习者说"懂了"，比一块内容讲完学习者懵了要好得多。

---

## 分块标准

### 一块内容的合理范围
- **最小**：1个新概念的定义（1页）
- **最大**：3页相关内容，或者一个完整的小主题
- **绝对上限**：5页。超过5页必须拆分，无论内容多连贯

### 什么情况必须拆分
1. 出现了3个以上新数学符号
2. 涉及超过1个新定理
3. 有例子页时，例子和理论必须分开（或合并成独立块）
4. 证明超过4步

### 什么情况可以合并
1. 定义页 + 直接对应的例子页（2页合1块）
2. 弱替代和强替代定义（概念紧密相关，但要分开解释）
3. 同一定理的陈述和证明（可合并，但证明要单独处理）

---

## 扫描分块流程

拿到一章内容后，先做以下步骤：

**第一步：** 扫描所有页面标题，识别自然的主题边界

**第二步：** 对每个主题估计：
- 包含几个新概念？
- 有没有例子页？
- 有没有证明？

**第三步：** 按上面的标准划分，然后**先告诉学习者分块方案**，再开始讲

**第四步：** 分块方案的呈现格式：
```
这一节内容我分成X块：

第一块（第N-M页）：[主题名] — [一句话说这块在讲什么]
第二块（第N-M页）：[主题名] — [一句话说这块在讲什么]
...

我们先从第一块开始。
```

## 已验证的分块案例（凸优化第四、五章）

以下是在实际教学中验证过的分块方式，可直接参考：

### 第四章
- 优化问题三要素（89页）：1块
- 可行点和最优点定义（90页）：1块
- 局部vs全局最优（91页）：1块
- 四个例子（92页）：1块（原文有图，让学习者看原图）
- 隐式vs显式约束（93页）：1块，重点讲对数障碍函数的直觉
- 标准形凸优化问题（95页）：1块
- 可行性问题（94页）：1块
- LP（101-104页）：整合为1块，含三个例子
- QP/QCQP（105-108页）：分成两小块（QP定义+最小二乘 / 风险规划+QCQP）
- SOCP（109-112页）：分成两小块（SOCP定义 / 随机鲁棒方法）
- SDP（113-116页）：分成两小块（锥形式+SDP定义 / LP和SOCP作为SDP特例）
- 变换问题（118-128页）：分成三小块（换变量换函数 / 消去引入松弛变量上图形式 / 凸松弛）
- DCP/CVXPY（130-132页）：1块
- 几何规划（134-136页）：1块
- 多目标优化（144-153页）：分成四小块

### 第五章
- 拉格朗日函数（156页）：1块
- 对偶函数（157页）：单独1块（这页很难，经常需要重讲）
- 四个例子（158-162页）：1块，逐个走推导流程
- 对偶问题定义（164-165页）：1块
- 弱对偶和强对偶（166-167页）：1块
- LP和QP对偶（168-169页）：1块
- 几何解释（170-171页）：1块
- 互补松弛（173页）：单独1块（推导容易懵，需要细讲）
- KKT条件（174-175页）：1块
- 敏感性分析（177-179页）：分成两小块（全局 / 局部）
- 问题重新表述（181-183页）：1块
- 替代定理（185-192页）：分成三小块（概念 / Farkas引理 / 无套利）

## 节奏控制要点

讲完每块后：
- **不要自动讲下一块**
- 说"说'继续'就进入下一块"
- 等待学习者明确表示理解

如果学习者在一块内容上卡了超过2次重讲：
- 退到更基础的前置知识
- 用具体数字例子走一遍
- 确认前置知识理解后再回来
